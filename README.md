# 노션 뽀모도로 레코더

노션 DB에 뽀모도로 개수를 기록하기 위한 용도의 일렉트론 앱

## 사용 방법

### 노션 연동 시
1. [notion API KEY 발급](https://www.notion.so/my-integrations)
2. notion DB 페이지 생성, 생성한 API 연결
3. notion DB 페이지에 Created Time 프로퍼티 추가
4. 노션 뽀모도로 레코더 앱 실행 API KEY, DB ID를 입행
5. 뽀모도로 앱 사용

### 뽀모도로 앱으로만 사용 시
1. 노션 뽀모도로 레코더 앱 실행
2. 우측 하단 '그냥 쓰기' 버튼 선택
3. 뽀모도로 앱 사용
